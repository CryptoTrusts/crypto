E) Online Trade Installation - top

To install Online Trade on online server, follow the steps below:

  1. Ensure your server supports PHP 7.0 and above and mysql 5.X
  2. Upload and extract project files to your desired directory.
  3. Create your database and import the SQL file in Online Trade root folder.
  4. Update .env file with your database information.
  5. Copy the content of the public folder to root folder and delete public folder.
  6. Open index.php file and edit following line to of code from require __DIR__.'/../bootstrap/autoload.php'; to require __DIR__.'/bootstrap/autoload.php';. Do thesame to other place where that code appear. (Usually two places). Congratulations ! you are done.
